================================================================================

SEPARATED FLOWS:

Client Flow:
  /client/intake → /client/dashboard
  /api/client/* (15 endpoints)
  Own data only

Admin Flow:
  /admin/dashboard → /admin/sessions
  /api/admin/* (25+ endpoints)
  All data access

  ✗ Create intake
  ✓ View own intake
  ✓ View all intakes
  ✓ Generate summary
  ✓ Add notes
  ✓ Assign cases
  ✓ Manage team (admin/manager only)
